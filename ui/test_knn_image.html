Test k-Nearest Neighbors visual similarity using AWS Bedrock multimodal embeddings on property images

About kNN Image: kNN Image uses multimodal embeddings to find visually similar images based on content and style. Uses AWS Bedrock amazon.titan-embed-image-v1 model to convert images into 1024-dimensional vectors in the same semantic space as text, then computes cosine similarity.
